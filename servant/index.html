Servant Help

Get started with your own little servants

Introduction


What is Servant?

Servant allows you to create things called servants, which run when the conditions you've set are met.

It's like having little things which do work for you.

The conditions and actions are pretty simple, English sentences such as "When I'm on the dashboard, disable XNeko
and display notification 'XNeko is gone!'". You can also write and use Javascript code.

Get Started

To get started, go to XKit Control Panel > Servant and click on "Add Servant."

Using In-Servant Tips

Each cause and action has a small "i" icon next to them. Select a cause or action, then hover over the icon to see what they do.

How to use Javascript code?


If you are a poweruser, you can also use Javascript code.

Using Javascript code allows you to do things Servant does not (yet) support.
You can access the XKit API, and will have access to jQuery.
Be careful though, and never copy/paste code from untrustworthy sources!
You should never use code you didn't review. Do not use obfuscated code.

Javascript on causes

When writing a cause Javascript code, simply return true or false and Servant will do the rest.
// Check if we should run.
if (XKit.interface.where().inbox === true) {
	return true;	
} else {
	return false;	
}
You can also use the Cause element to "initialize" your extension, for example, add a control button:
XKit.interface.create_control_button("myservant", "data:image/png;base64,iVBOR....", 
    "My Button!", function() {
        alert("hello world!");
});
if (XKit.interface.where().inbox === true) {
	return true;	
} else {
	return false;	
}

Javascript on actions

If your Servant runs on a post, you can access the post using the variable post. 
// Make the said post's background red!
$(post).css("background","red");
You can also use the "cause return"s:
// Make the said post's background red!
$(post).css("background","red");
$(post).append("I made this red because it contained \"%1\"!");
