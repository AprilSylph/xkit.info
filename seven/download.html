Download XKit

Installing XKit is as easy as 1-2-3. Really.

Thanks for choosing XKit!

I've identified your browser as Google Chrome.

 

1. Very Important!

Before beginning, please read the following.

  • If you have XKit 6 or below, XKit Next or XKit 7 Preview Beta installed, you have to uninstall them.
  • Disable or uninstall Greasemonkey, Stylish and other Tumblr extensions before installing XKit.
    I've seen so many people complaining about XKit not working, only for them to say that it started working when they've disabled or removed the other dashboard themes or other Tumblr extensions they had installed. Some extensions/themes cause problems with XKit, some remove elements and scripts XKit require in order to function properly, causing conflicts. If you are dependent on an extension, please enable it after using XKit for a while, to see if it is it or XKit itself is causing the problem. When re-enabling them, do it one-by-one, checking your dashboard to determine which extension might be causing the problem.
  • If you do not know how to remove extensions, please do not install them.
To learn how to uninstall/disable extensions, please go to the XKit FAQ or check your browser's help pages:
Chrome  |  Firefox  |  Safari  
 

2. Start Installation

Go to Web Store.

Google Chrome does not allow extensions to be installed from outside of Chrome Web Store.
So, click on the button below to go to the Google Web Store and install XKit there.
Go to Web Store
 

3. And you are done!

Go to your dashboard to finish installation.

After you install XKit from Google Web Store, go back to your dashboard.
XKit will download the required packages, and install itself.
Go to Tumblr
 

Something went wrong?

If something went wrong while installing XKit, even after disabling other extensions,
click on the button below to run XKit Troubleshooting Tool.
You can also check the FAQ, under the Installing + Upgrading tab.
